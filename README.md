### 🧠 Sobre o projeto

Projeto acadêmico desenvolvido com **Blender**, com o objetivo de reconstruir digitalmente o ambiente da faculdade em um **tour virtual interativo em 360°**.

A proposta envolve modelagem 3D, organização de cena e criação de uma experiência imersiva, permitindo a navegação por um campus virtual como se fosse um ambiente real.

---

### 🛠️ Tecnologias utilizadas

- Blender (modelagem e organização 3D)  
- Renderização de panoramas 360°  
- Plataforma de tour virtual (Meu Passeio Virtual)  
- Preparação e exportação de cenas para visualização web  

---

### 🎯 Objetivo

- Simular um ambiente educacional em 3D  
- Aplicar conceitos de modelagem e composição de cena  
- Criar uma experiência de navegação imersiva  
- Explorar fluxos de trabalho de visualização interativa
